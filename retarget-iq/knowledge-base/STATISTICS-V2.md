## Methodology Documentation

### V2 Extraction Process

**Step 1: Source Review**
- Read actual email threads (not just extracted text)
- Read full call transcripts (not just snippets)
- Understand conversation context and flow

**Step 2: Question Identification**
- Identify genuine product/service questions
- Skip scheduling, greetings, internal coordination
- Look for patterns across multiple customers

**Step 3: Reformulation**
- Convert conversational questions to professional format
- Replace pronouns with specific referents
- Remove personal context and names
- Make universally applicable

**Step 4: Answer Creation**
- Extract authoritative answer from email/call
- Clean email artifacts (signatures, metadata)
- Make complete and self-contained
- Add specifics (prices, timelines, processes)

**Step 5: Categorization**
- Assign to logical category
- Add relevant keywords for search
- Reference source for verification
- Assign unique ID

**Step 6: Quality Control**
- Verify answer accuracy
- Check for completeness
- Ensure professional tone
- Validate JSON structure
